## About the Project
This repository contains the Part One HTML skeleton and planning documentation
for the EMT Bakery website – a professional bakery serving the general public,
corporate clients, and wedding couples in Johannesburg, South Africa.

The website provides:
- Online product showcase (artisan breads, custom cakes, wedding cakes)
- Online ordering system
- Gallery of past work
- Contact and enquiry forms

## HTML Standards Applied

All HTML files in this project adhere to:

- **HTML5 Semantic Elements** – `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`, `<figure>`, `<figcaption>`, `<address>`
- **WCAG 2.1 AA Accessibility** – ARIA landmark roles, `aria-label`, `aria-current`, `aria-describedby`, skip links, proper label/input associations
- **SEO Best Practices** – Unique `<title>` per page, meta description, canonical links, Open Graph tags, semantic heading hierarchy (H1→H2→H3)
- **Mobile-First Foundation** – Viewport meta tag, content structured for single-column mobile layout
- **Valid HTML5** – All files comply with W3C HTML5 specification
